# 📊 YouTube 분석기

YouTube 영상을 키워드로 검색하고 품질과 인기도를 분석하는 웹 애플리케이션입니다.

## ✨ 주요 기능

- 🔍 **키워드 기반 검색**: 원하는 키워드로 YouTube 영상 검색
- 📊 **상세 통계**: 조회수, 좋아요, 댓글 수 등 주요 지표 표시
- 🎯 **스마트 정렬**: 품질 점수, 조회수, 좋아요, 댓글 수, 최신순 정렬
- 💡 **인사이트 분석**: 각 영상별 상세 인사이트 및 품질 점수 제공
- 📈 **채널 통계**: 채널 구독자 수 및 영상 수 표시
- 🎨 **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원

## 🚀 시작하기

### 사전 요구사항

- Node.js (v16 이상)
- YouTube Data API v3 키

### YouTube API 키 발급

1. [Google Cloud Console](https://console.cloud.google.com/)에 접속
2. 새 프로젝트 생성 또는 기존 프로젝트 선택
3. "API 및 서비스" > "라이브러리"로 이동
4. "YouTube Data API v3" 검색 및 사용 설정
5. "사용자 인증 정보" > "사용자 인증 정보 만들기" > "API 키" 선택
6. 생성된 API 키 복사

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 미리보기
npm run preview
```

## 📖 사용 방법

1. **API 키 설정**
   - 애플리케이션 실행 후 YouTube Data API v3 키 입력
   - API 키는 브라우저 로컬 스토리지에 안전하게 저장됩니다

2. **키워드 검색**
   - 검색창에 원하는 키워드 입력 (예: "React Tutorial", "유튜브 마케팅")
   - 검색 버튼 클릭 또는 Enter 키 입력

3. **결과 확인**
   - 검색 결과를 다양한 기준으로 정렬
   - 각 영상 카드 클릭시 YouTube에서 바로 시청
   - "인사이트 보기" 버튼으로 상세 분석 확인

## 🎯 품질 점수 계산

품질 점수는 다음 요소들을 종합하여 0-100점으로 계산됩니다:

- **조회수** (최대 30점): 영상의 도달 범위
- **참여율** (최대 30점): (좋아요 + 댓글) / 조회수
- **좋아요 비율** (최대 20점): 좋아요 / 조회수
- **댓글 활성도** (최대 20점): 댓글 / 조회수

### 성능 등급

- 🟢 **최고** (80-100점): 매우 높은 품질의 콘텐츠
- 🔵 **우수** (60-79점): 우수한 품질의 콘텐츠
- 🟡 **양호** (40-59점): 양호한 품질의 콘텐츠
- 🔴 **보통** (20-39점): 평범한 품질의 콘텐츠
- ⚪ **낮음** (0-19점): 개선이 필요한 콘텐츠

## 🛠️ 기술 스택

- **Frontend**: React 18
- **Build Tool**: Vite
- **Styling**: CSS3 (반응형 디자인)
- **API**: YouTube Data API v3
- **HTTP Client**: Axios

## 🔒 개인정보 보호

- API 키는 브라우저의 로컬 스토리지에만 저장됩니다
- 서버로 전송되지 않으며 사용자의 브라우저에서만 관리됩니다
- 언제든지 API 키를 변경하거나 제거할 수 있습니다

## ⚠️ 주의사항

- YouTube Data API v3는 일일 할당량이 있습니다 (기본 10,000 units/day)
- 각 검색은 약 100 units를 사용합니다
- 할당량 초과시 다음 날까지 API 사용이 제한될 수 있습니다
- [할당량 관리 페이지](https://console.cloud.google.com/apis/api/youtube.googleapis.com/quotas)에서 사용량 확인 가능
